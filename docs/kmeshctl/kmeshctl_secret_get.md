## kmeshctl secret get

Get IPsec key and configuration by kmeshctl

```bash
kmeshctl secret get [flags]
```

### Examples

```bash
# Get IPsec key and configuration by kmeshctl. The results will be displayed in JSON format.
kmeshctl secret get
```

### Options

```bash
  -h, --help   help for get
```
